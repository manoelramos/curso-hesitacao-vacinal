2

Qual é uma vantagem da vacinação para a saúde pública?

3

4

5

6

Julgue as afirmações abaixo como verdadeiras (V) ou falsas (F):

( ) O principal objetivo das vacinas é proteger o indivíduo e a comunidade contra doenças graves.

( ) Imunidade de grupo ocorre quando uma população vacinada reduz a circulação de uma doença, protegendo também os não vacinados.

( ) As vacinas atenuadas contêm agentes infecciosos vivos enfraquecidos, que não causam doença grave.

( ) As vacinas de ácidos nucleicos utilizam microrganismos vivos para estimular o sistema imunológico.
